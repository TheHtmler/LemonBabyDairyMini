① 减法：删 / 移走的

  • 「更多」整张卡删除：营养设置 / 食物库 / 用药管理 / 数据共享 在「我的」Tab 已逐字重复。
  • 「近7天趋势」移走：归「报告分析」Tab，首页只留一个入口。
  • 「今日时间轴」移走：归「数据记录」Tab，首页只留一个入口。
  • 快捷记录 7 → 5：去掉「检测报告」(本就是 Tab)、「量身高」(并入营养卡 ✎)。

    • ② 加法 / 调整 提升使用率

  • 今日照护完成度环：喂奶 / 用药 / 体重 合成一个一眼概览，家长有「想点满」的动力。
  • 快捷记录前置：从第 4 张卡提到「热量环」正下方（第 3 位），一进首页就好记，不用滚到底。
  • 连续记录天数（streak）已砍：漏记≠没护理，价值低且易带挫败感。
  • "妈妈/爸爸"协作已砍：现有角色只有 创建者/参与者，无父母角色；如需可后续用真实昵称做"最近由 XX 记录"。

    • ③ 结果

  • 首页从 8 卡片 → 焦点卡 + 4 张卡，竖向长度约缩短 1/3。
  • 记录入口前置，首页"能记一笔"的主任务更突出。
  • 四个 Tab 职责清晰：今日(做) / 数据记录(查) / 报告分析(看趋势) / 我的(设置)。
  • 风险低：减法部分改动很小；完成度环可纯函数实现并补单测（已有 home-dashboard.test.js 位）。
