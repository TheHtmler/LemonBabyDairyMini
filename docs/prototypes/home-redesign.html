为什么这样改？

  • 消除重复：旧首页几乎是「数据记录」的今日版（同样的汇总卡 + 同样的喂奶/食物/用药/治疗列表 + 同样的弹窗）。
  • 首页 = 看板：只放「今天要看一眼的东西」和「最快下手的入口」。
  • 有用的数据：营养达成环 + 进度条（对比目标，不只是数字）、喂养进度与下一顿建议、近7天体重/蛋白趋势、连续记录天数。
  • 用药打卡新：按规律自动列出今日该吃的药（精氨酸/左卡×2/AD·D3交替/B12隔日），可勾选，独一份。
  • 分类入口：6 个快捷记录大按钮 + 更多功能宫格，一步直达。
  • 数据记录页保持不变：负责任意日期的逐条明细查看 / 补录 / 复制。
