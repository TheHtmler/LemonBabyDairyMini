# MMA 临床思考框架

用于组织回答，不替代最新指南或专科医嘱。

## 1. 先分场景

### 新生儿期

关注：

- 新生儿筛查阳性后是否已完成确诊路径
- 是否出现拒奶、呕吐、嗜睡、呼吸异常、张力改变
- 喂养间隔是否过长，是否存在禁食风险
- 家属是否已获得代谢专科的初始应对方案

优先输出：

- 是否存在紧急评估信号
- 在专科接手前只能讨论的一般喂养原则
- 哪些信息必须尽快由本地团队确认

### 辅食期

关注：

- 辅食添加节奏是否过快或结构失衡
- 单种高蛋白食物是否超出全天管理范围
- 家属是否只盯单个食物，忽略全天总量
- 进食行为和接受度是否影响能量摄入

优先输出：

- 食物判断框架
- 更稳妥替代品思路
- 如何与全天喂养计划衔接

### 稳定期

关注：

- 生长发育是否达标
- 摄入是否足够，是否存在挑食、拒食、辅食结构失衡
- 天然蛋白与特殊配方之间是否平衡
- 是否有反复轻度失代偿、贫血、肾功能或神经发育问题

### 急性风险期

关注：

- 发热、感染、禁食、呕吐、腹泻、摄入下降
- 嗜睡、意识变化、抽搐、脱水
- 是否已有代谢危机既往史

这类场景先强调紧急评估与专科联系，不先展开食谱细节。

### 长期随访期

关注：

- 身高体重和头围趋势
- 认知、运动、语言和行为发育
- 肾脏、血液学和营养相关长期问题
- 家庭执行压力、外出、入园和照护者交接

优先输出：

- 长期管理目标
- 随访与监测主题
- 家属可执行的日常管理框架

## 2. 喂养问题的回答顺序

1. 当前是否稳定
2. 能量是否足够
3. 蛋白管理是否在已知预算内
4. 特殊配方、天然食物、辅食之间如何分配
5. 进食行为和家庭执行是否可持续
6. 是否需要化验和专科随访支持

## 3. 对食物问题的默认拆解

当用户问“能不能吃某食物”时，至少拆成：

- 食物的蛋白负荷和份量
- 当前年龄与喂养阶段
- 是否稳定期
- 是否能纳入全天总量管理
- 有没有更稳妥替代品

## 4. 长期管理时常见主题

- 营养不良或生长迟缓
- 过度限制导致的能量或微量营养素不足
- 神经发育和康复支持
- 肾脏和血液学并发症监测
- 家庭教育、感染期预案、外出喂养和幼托场景

## 5. 输出边界

- 可以提供结构化医学信息、家属教育框架、问题清单、随访思路。
- 不应在缺少临床资料时给出个体化处方、具体剂量或替代专科团队的危机处理方案。

## 6. 四类默认回答骨架

### 新生儿期骨架

1. 是否存在急症信号
2. 当前只能确认的一般原则
3. 喂养间隔与禁食风险
4. 需要本地团队立即确认的项目
5. 证据或流程来源

### 辅食期骨架

1. 当前稳定性
2. 该食物或食材的判断逻辑
3. 如何纳入全天管理
4. 可替代选择
5. 何时需要化验或专科复核

### 感染应激期骨架

1. 是否需要立刻联系急诊或代谢团队
2. 哪些是高风险信号
3. 一般性 sick-day 原则
4. 不能远程替代的部分
5. 需要核验的最新指南或中心预案

### 长期随访期骨架

1. 当前目标是生长、发育还是减少失代偿
2. 饮食结构与依从性重点
3. 长期并发症和监测主题
4. 家庭/托育/外出执行建议
5. 不确定点与专科确认事项
